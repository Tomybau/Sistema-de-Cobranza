## Structure

Each automation lives in its own folder under a category:

```
backend/
├── scraping/
├── outreach/
├── datos/
└── etl/
```

Each automation folder contains: `TASK.md`, `main.py`, `requirements.txt`.
Follow the `automation-builder` global skill when creating new automations.
